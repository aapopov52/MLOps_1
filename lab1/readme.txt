data_creation.py - создаёт набор данных, включающий в себя:
4 признака fut1, fut2, fut3, fut14 - распределнны по нормальному закону
колонку с "аномалиями" - для добавления шумов - распределён по нормальному закону
целевая переменная - target, определяемый по формуле: 0.1 + fut1 * 2 + fut2 * 3 + fut3 * 0.5 + fut4 * 2.5
посредством колонки с аномалиями корректруем значения признаков fut2 и fut3 - "ломаем чёткую линейную зависимость

разбиваем выборку на тренировочную и тестовую train_df, test_df, сохраняем в соответствующих папках .


model_preprocessing.py
производим предварительную подготоквку данных, состоящую в стандартизации данных.
Обучаемся на основе тренировочных данных: получаем среднее и сигму.
С помощью среднего и сигмы преобразуем ка данные тренировочной вборки, так и данные тестоовй выборки.
сохроаем преобразованные данные в папки train и test соответственно.


model_preparation.py
обучет модель линейной регресии на основе тренировочной выоборки (train/train1.csv).
Сохзраняет можель в файл model/model.pkl.

model_testing.py
проверяем работу модели на тестовых данных: загружаем модель из файла model/model.pkl, получаем предсказания, сравниваем с targetom, рассчитываем параметры:
mse, mae, r2. выводим резульат на экран.

pipeline.sh
Запуск всего описанного выше безобразия на выборление.
